nagios.vim
=============

Description
-----------

[Vim](http://www.vim.org/) plugin for [nagios](http://www.nagios.org)

This plugin provides:

  * Syntax highlight
  * UltiSnips snippets


Installation
------------

Use [Vundle](https://github.com/gmarik/vundle), to install this plugin:

    # inside ~/.vimrc
    :Plugin 'crazy-canux/nagios.vim'
