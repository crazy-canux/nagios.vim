# nagios.vim

[nagios](https://www.nagios.org) syntax for [vim](http://www.vim.org/).

Inspired by <https://github.com/pld-linux/vim-syntax-nagios>

# How to install

## pathogen

    cd ~/.vim/bundle
    git clone git@github.com:crazy-canux/nagios.vim.git

## vundle

    vi ~/.vimrc
    Plugin 'crazy-canux/nagios.vim'
