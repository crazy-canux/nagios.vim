mv-vim-nagios
=============

Description
-----------

[Vim](http://www.vim.org/) plugin for [Icinga](http://www.icinga.org)

This plugin provides:

  * Syntax highlight
  * Snipmate snippets


Installation
------------

Use [Vundle](https://github.com/gmarik/vundle), to install this plugin:

    # inside vim
    :BundleInstall 'mv/mv-vim-icinga'

    # Add to ~/.vimrc
    Bundle 'mv/mv-vim-icinga'
